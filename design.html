My Designs.

Below are some of my current designs which I am most proud of. These designs were made from 2023 to now. I'm continuously improving, so this gallery continues to expand. Stay tuned!

Lost In Space.

Photo Composite I created, featuring a mix of an astronaut with a space background. In the astronaught's visor's, there is a mixed image of a 1950's style picture, taken approximately around that time. The theme for this was: Lost in space during the 1950s.

Technology in the 90s.

Another photo composite I created, featuring an old macintosh computer, with an older client of Adobe Photoshop running on it. The theme for this was: Interesting new technology in the 1990s.

Stream Offline.

A mockup screen for a twitch.tv, this one features text that says "Stream Offline, Thanks for Watching". There is a contour style of waves on the left side, and a wireframe with a couple warps on the right side of the screen.

Fallout Thumbnail.

This was a post modernist style take on what a thumbnail for the video game: Fallout, would look like. There isn't exactly one set theme for this thumbnail, but rather more so as a proof of concept, that I can further develop if needed.

Enigma.

A more experimental graphic, for this one, I was experimenting with different fonts, and a mix of Japanese text. I also tried to incorporate more text on the page, to give it a style, and to make it seem more enticing. The more details, the more interesting the graphic, is sort of what I thought about when making this one.

Pursue.

Another experimental graphic, though this is the type I might actually want to implement onto a piece of clothing. The main theme for this one is 'Pursue', so it features a definition of pursue, as well as Japanese text for a more graphic feel. For this one, I used a color that I usually don't incorporate into my designs, that being magenta/fuschia.

Experimental Album Cover.

This is an experimental album cover, featuring my name rather than a band name. I used a variety of shapes to give off an interesting geometric style to this cover, as well as combining multiple different vibrant colors to make the graphic look more interesting. There is also a lot of Japanese text around the piece, to create some more interesting graphic elements on the cover.

Porsche 911.

This is a graphic I made for a photo I took of a Porsche 911 car in Italy. The word Porsche is repeated 5 times, 4 of them are hollow, 1 of them is full with color. There is more Japanese text to create a streetwear style graphic. The background also features a contour style damascus pattern.

diaL advertisement.

This graphic is a hypothetical header graphic for my brand, if it were to be an actual clothing brand. This graphic features text that describes a new clothing drop, as well as some quirky Y2K elements mixed in there to give it a cluttered feel. It also features a hypothetical discount code for 15% off the first order.
